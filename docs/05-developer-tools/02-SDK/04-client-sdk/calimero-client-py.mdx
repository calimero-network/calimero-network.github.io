# Python Client SDK

The Calimero Python Client SDK provides a comprehensive, high-performance
interface for interacting with Calimero Network APIs. Built with Rust and PyO3,
it offers native performance with the ease of Python development.

## Overview

The Python client library is designed for developers who want to build Calimero
applications using Python, whether for web development, data analysis,
automation, or integration with existing Python ecosystems.

### Key Features

- **üöÄ High Performance**: Built with Rust and PyO3 for optimal performance
- **üîí Type Safety**: Strongly typed Python bindings with full IDE support
- **‚ö° Async Support**: Built-in async/await support for concurrent operations
- **üåê Multi-Protocol**: Support for NEAR, Ethereum, ICP, and Starknet
- **üì¶ Easy Installation**: Simple `pip install` with no complex dependencies
- **üîß Comprehensive API**: Full access to all Calimero Network functionality

## Supported Protocols

- ‚úÖ **NEAR Protocol** - Account-based architecture with optimized storage
- ‚úÖ **Ethereum** - Full EVM compatibility with gas optimization
- ‚úÖ **Internet Computer Protocol (ICP)** - Canister-based execution model
- ‚úÖ **Starknet** - Cairo-based smart contracts with account abstraction

## Quick Start

### Installation

Install the Python client from PyPI:

```bash
pip install calimero-client-py
```

### Basic Usage

```python
import asyncio
from calimero_client_py import create_connection, create_client, AuthMode

async def main():
    # Create a connection
    connection = create_connection(
        base_url="http://localhost:2528",
        auth_mode=AuthMode.NONE
    )

    # Create a client
    client = create_client(connection)

    # List available contexts
    contexts = await client.list_contexts()
    print(f"Found {len(contexts)} contexts")

    # List applications
    apps = await client.list_applications()
    print(f"Found {len(apps)} applications")

if __name__ == "__main__":
    asyncio.run(main())
```

## Architecture

### Rust + PyO3 Foundation

The Python client is built on a solid Rust foundation using PyO3, which
provides:

- **Native Performance**: Rust core ensures optimal speed
- **Memory Safety**: Rust's ownership system prevents memory leaks
- **Python Integration**: Seamless Python bindings with PyO3
- **Cross-Platform**: Works on Windows, macOS, and Linux

### Async-First Design

The client is designed with async/await in mind:

```python
# All operations are async
async def example():
    client = create_client(connection)

    # Concurrent operations
    contexts_task = client.list_contexts()
    apps_task = client.list_applications()

    contexts, apps = await asyncio.gather(contexts_task, apps_task)
    return contexts, apps
```

- **Native integration** with Python data science libraries

### Benchmark Example

```python
import time
import asyncio
from calimero_client_py import create_connection, create_client

async def benchmark():
    connection = create_connection("http://localhost:2528")
    client = create_client(connection)

    start = time.time()

    # Execute 100 concurrent operations
    tasks = [client.list_contexts() for _ in range(100)]
    await asyncio.gather(*tasks)

    end = time.time()
    print(f"100 operations completed in {end - start:.2f} seconds")

asyncio.run(benchmark())
```

## Integration Examples

### With Merobox

```python
import asyncio
from calimero_client_py import create_connection, create_client
from merobox import Cluster

async def deploy_with_merobox():
    # Start Merobox cluster
    with Cluster() as cluster:
        # Get node URL
        node_url = cluster.get_node_url("node-1")

        # Create Calimero client
        connection = create_connection(node_url)
        client = create_client(connection)

        # Deploy application
        app = await client.install_application(
            url="https://example.com/app.wasm",
            metadata=b'{"name": "My App"}'
        )

        print(f"Deployed app: {app['application_id']}")
```

### With FastAPI

from calimero_client_py import create_connection, create_client

async def benchmark():

### Custom Error Handling

```python
from calimero_client_py import ClientError, AuthError, NetworkError
import logging

logger = logging.getLogger(__name__)

async def robust_operation(client, operation, *args, **kwargs):
    """Execute operation with robust error handling and retries."""
    max_retries = 3
    retry_delay = 1

    for attempt in range(max_retries):
        try:
            return await operation(*args, **kwargs)
        except NetworkError as e:
            if attempt < max_retries - 1:
                logger.warning(f"Network error, retrying in {retry_delay}s: {e}")
                await asyncio.sleep(retry_delay)
                retry_delay *= 2
            else:
                logger.error(f"Network error after {max_retries} attempts: {e}")
                raise
        except AuthError as e:
            logger.error(f"Authentication error: {e}")
            raise
        except ClientError as e:
            logger.error(f"Client error: {e}")
            raise
```

## Migration Guide

### From JavaScript Client

If you're migrating from the JavaScript client:

```python
# JavaScript (old)
import { createConnection, createClient } from '@calimero-network/calimero-client';

const connection = createConnection('http://localhost:2528');
const client = createClient(connection);

# Python (new)
from calimero_client_py import create_connection, create_client

connection = create_connection("http://localhost:2528")
client = create_client(connection)
```

### Key Differences

1. **Async/Await**: Python client uses native async/await
2. **Error Handling**: More specific exception types
3. **Type Safety**: Better type hints and IDE support
4. **Performance**: 2-3x faster for CPU-intensive operations

### Migration Checklist

- [ ] Update import statements
- [ ] Convert callback-based code to async/await
- [ ] Update error handling to use specific exception types
- [ ] Test all functionality with new client
- [ ] Update documentation and examples
