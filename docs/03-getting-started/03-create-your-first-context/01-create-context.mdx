## Create context with Admin Dashboard

1. Navigate to the “Contexts” tab press "Start new context" button.
2. Select app from the list using "Browse" button.
3. Select "Template Application"
4. Press "Start" to initiate context creation with the selected app.

Once the context is created, you will be redirected to the context dashboard.
You will need context ID later so don't close page yet.

As currently coordinator node is required to be inside each context, we need to
invite him to participate. Continue with the
[Invite To Context](./02-invite-to-context.mdx) guide.

## Create context with CLI using meroctl

```bash
cargo run -p  meroctl -- --node-name node1 --home data app install --url {url} --metadata '{“contractAppId”: {contractApplicationId} }'
```

You can find the application url and contractApplicationId the application
details in the Admin Dashboard.

![Application details](/admin-dashboard/application-details-for-context-creation.png)

```bash title="Example"
cargo run -p  meroctl -- --node-name node1 --home data app install --url https://blobby-public.euw3.prod.gcp.calimero.network/bafkreihl5o6etrnpy7dlgixz3onbfb3og4dll2yqsqaebqakuldk6e2qya --metadata '{“contractAppId”: “980265ba072119a9074e429dea477e1c084a94e8d9f645c0158680e6942fb99e” }'
```

After the application is installed, you can create a new context with the
application.

```bash
cargo run -p meroctl -- --node-name node1 --home data  context create --application-id {application id}
```

```bash title="Example"
cargo run -p meroctl -- --node-name node1 --home data  context create --application-id A357tXwvVU14wikYLwoy4e1FHEXDynxtsXLpqbHRGmFw23w95XmV2fzqeWjFodk6bRwKTHigsJvFh67U5EtqbKQC
```

As currently coordinator node is required to be inside each context, we need to
invite him to participate. Continue with the
[Invite To Context](./02-invite-to-context.mdx) guide.
