### Install the Application

Run the following command to install the application:

```bash title="Node Terminal"
application install url {url} {metadata}
```

:::note

metadata example:

```bash
'{"contractAppId": "3Hfk2VekXQ58vYHW3hUtA3mh2Rwtb1brV1RKEXtfvfsf"}'
```

:::
